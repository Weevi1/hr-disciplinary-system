## Layout Specifications

### Page Setup
- **Format:** A4 Portrait (210mm × 297mm)
- **Margins:** 20mm on all sides
- **Border:** Dual-layer decorative border (2mm outer, 0.5mm inner)
- **Corner Decorations:** 5mm colored squares in each corner

### Typography Hierarchy

| Element | Font | Size | Style | Color |
|---------|------|------|-------|-------|
| Main Title | Helvetica | 28pt | Bold | Black |
| Recognition Badge | Helvetica | 12pt | Bold | White on colored bg |
| Employee Name | Helvetica | 24pt | Bold | Green (#16a34a) |
| Achievement Title | Helvetica | 16pt | Bold | Black on orange bg |
| Section Headings | Helvetica | 12pt | Bold | Green |
| Body Text | Helvetica | 11pt | Normal | Dark gray |
| Italics | Helvetica | 11pt | Italic | Gray |
| Footer | Helvetica | 8pt | Normal | Light gray |

### Color Scheme (Default Green Theme)

- **Primary:** #16a34a (Green - used for name, headings, skill tags)
- **Secondary:** #15803d (Dark Green - used for borders)
- **Accent:** #f59e0b (Orange - used for achievement highlight, badge)
- **Background:** White
- **Text:** Black (#000000) and Gray (#404040, #505050, #787878)

### Spacing Guidelines

- **After Logo:** 10mm
- **After Title:** 12mm
- **After Badge:** 15mm
- **Between Sections:** 8-10mm
- **Line Height:** 6mm (body text), 7mm (titles)
- **Paragraph Spacing:** 5mm

## Design Features Explained

### 1. **Decorative Border**
The dual-layer border creates an elegant frame:
- **Outer Border:** 2mm thick, primary color (green)
- **Inner Border:** 0.5mm thick, accent color (orange)
- **Corner Squares:** 5mm accent-colored squares at each corner
- **Purpose:** Gives certificate a professional, frameable appearance

### 2. **Recognition Type Badge**
Color-coded badge indicates achievement category:
- **Shape:** Rounded rectangle (2mm border radius)
- **Colors:**
  - Achievement = Orange
  - Excellence = Purple
  - Innovation = Blue
  - Leadership = Pink
  - Service = Green
  - Teamwork = Light Green
- **Text:** White, uppercase, centered
- **Purpose:** Quick visual identification of recognition type

### 3. **Employee Name Emphasis**
The most prominent element on the certificate:
- **Size:** 24pt (largest text on page)
- **Color:** Green (brand color)
- **Style:** Bold, uppercase, underlined
- **Position:** Centered
- **Purpose:** Employee is the hero of this certificate

### 4. **Achievement Title Highlight**
Visual emphasis on the specific achievement:
- **Background:** Semi-transparent orange box
- **Text:** 16pt bold black
- **Alignment:** Centered with word wrapping
- **Purpose:** Draws eye to the specific accomplishment

### 5. **Business Impact Section**
Quantified results in highlighted box:
- **Background:** Light green (#16a34a with 10% opacity)
- **Border:** 0.3mm green border with 2mm rounded corners
- **Text:** 10pt italic, dark gray
- **Purpose:** Shows measurable organizational benefit

### 6. **Skills Tags**
Visual tag system for demonstrated competencies:
- **Shape:** Rounded rectangles (1.5mm border radius)
- **Background:** Semi-transparent green
- **Border:** 0.2mm green border
- **Text:** 9pt normal, green
- **Layout:** Horizontal flow with automatic line wrapping
- **Purpose:** Clear visual summary of competencies

### 7. **Recognition Rewards**
Bulleted list of tangible rewards:
- **Font:** 10pt normal
- **Format:** Bullet points with clear labels
- **Types:** Money (R currency), Time off (days), Gifts
- **Purpose:** Documents formal recognition given

### 8. **Signature Section**
Formal authorization area:
- **Signature Space:** 70mm wide × 20mm high
- **Signature Line:** Thin black line (0.3mm)
- **Name:** 11pt bold
- **Title:** 10pt normal, gray
- **Date:** 9pt italic, gray
- **Purpose:** Official authorization and authenticity

### 9. **Organization Seal**
Optional official stamp/seal:
- **Position:** Bottom right corner
- **Size:** 30mm × 30mm
- **Format:** PNG with transparency
- **Purpose:** Additional official branding

### 10. **Footer Metadata**
Certificate tracking information:
- **Left:** Certificate Number (CERT-YYYY-NNNN)
- **Center:** System attribution and generation date
- **Right:** Recognition ID (truncated)
- **Font:** 8pt, light gray
- **Purpose:** Tracking, verification, system attribution

## Customization Options

Organizations can customize the following:

### Brand Colors
```typescript
branding: {
  colors: {
    primary: '#16a34a',    // Main brand color (headings, name)
    secondary: '#15803d',  // Supporting color (borders)
    accent: '#f59e0b'      // Highlight color (achievement title)
  }
}
```

### Logo & Seal
```typescript
branding: {
  logo: 'data:image/png;base64,...',           // Top center
  certificateSeal: 'data:image/png;base64,...' // Bottom right
}
```

### Recognition Types
Add custom recognition types with custom colors:
```typescript
recognitionType: 'custom',
customTypeLabel: 'SPECIAL RECOGNITION',
customBadgeColor: '#8B5CF6' // Purple
```

## Printing Recommendations

### Paper & Finishing
- **Paper:** 200-250gsm premium certificate paper
- **Finish:** Matte or linen texture recommended
- **Size:** A4 (210mm × 297mm)
- **Color:** Full color (CMYK)

### Printer Settings
- **Quality:** Best/High quality
- **Color Mode:** Color
- **Paper Type:** Heavy/Cardstock
- **Margins:** None (full bleed if available)

### Framing
- **Frame Size:** A4 standard frame (210mm × 297mm)
- **Mat Border:** Optional white mat with 30-40mm border
- **Glass:** Clear or non-glare glass
- **Backing:** Acid-free backing board

---

## File Output Specifications

### PDF Properties
- **Version:** PDF 1.4 (compatible with all readers)
- **Compression:** Enabled (reduces file size by ~30%)
- **Fonts:** Embedded Helvetica (standard font)
- **Images:** Embedded, optimized for print
- **File Size:** 150-500 KB (depending on images)

### Resolution
- **Text:** Vector (scalable)
- **Images:** 150-300 DPI recommended
- **Logo:** Vector preferred, or 300 DPI PNG
- **Signature:** 150 DPI sufficient
