## Data Flow

1. **User fills out fields** → Component validates in real-time
2. **Validation updates** → `onValidationChange(isValid)` called
3. **Data changes** → `onDataChange(formData)` called with current state
4. **User clicks Next** → Wizard checks `correctiveDiscussionValid` state
5. **If valid** → Advances to Delivery step with data
6. **If invalid** → Shows alert, stays on current step
